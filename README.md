# CTF-Writeups 收集

本仓库使用 GitHub Actions 自动浅克隆（depth=1）CTF 相关的仓库，包括 writeups 和 wiki 资源。

## 自动克隆

仓库配置了 GitHub Actions 工作流，会在以下情况自动运行：

- 推送到 main 或 master 分支时
- 创建 Pull Request 时
- 每周一自动运行
- 可手动触发
